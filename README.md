# Accordis

Interactive chord detection app with a piano keyboard UI.

## Features

- Select piano keys and detect matching chord(s)
- Shows detected results with optional translations
- Fast client-side chord logic
- Unit/integration tests with Vitest + Testing Library

## Tech Stack

- Next.js 16 (App Router)
- React 19 + TypeScript
- Tailwind CSS 4
- Vitest + jsdom + Testing Library

## Getting Started

```bash
npm install
npm run dev
